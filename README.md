# Интерактивная Доска (InteractBoard)

Веб-приложение для просмотра, фильтрации и управления каталогом досок. Приложение позволяет пользователям искать доски, добавлять их в избранное и просматривать подробную информацию о каждой доске.

## Возможности

- **Поиск** - быстрый поиск по названиям досок
- **Фильтрация** - фильтрация по категориям
- **Избранное** - сохранение любимых досок локально
- **Пагинация** - удобная навигация по результатам
- **Адаптивный дизайн** - работает на всех устройствах
- **Быстрая загрузка** - ленивая загрузка компонентов

## Технологический стек

- **React 19** - библиотека для UI
- **TypeScript** - типизированный JavaScript
- **Redux Toolkit** - управление состоянием
- **React Router 7** - клиентская маршрутизация
- **Axios** - HTTP клиент
- **SCSS** - препроцессор CSS
- **React Paginate** - компонент пагинации

## Требования

- Node.js (версия 14 или выше)
- npm или yarn

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/rostislavk636/interectboard.git
cd interectboard
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Запуск в режиме разработки

```bash
npm start
```

Приложение откроется в браузере по адресу `http://localhost:3000`

### 4. Сборка для продакшена

```bash
npm run build
```

Собранное приложение находится в папке `build/`

### 5. Развёртывание на GitHub Pages

```bash
npm run deploy
```
